## How This Is Organised

- **One file per concept.** No enormous catch-all files.
- **Libraries** get a flat file until they accumulate enough entries to earn their own subfolder.
- **AI Agents** is intentionally fluid — this space is evolving fast and the content here reflects current understanding, not settled doctrine.
- Every file follows the same template so entries are scannable and consistent.
